# AvalonApp
## Text to speech : 

Utilisation de l'outil TTSMP3 (https://ttsmp3.com/text-to-speech/French/)

PS: Oberon n'a aucun impact sur le texte généré.

Texte sans aucun rôles :
```
Chacun ferme les yeux et tend son poing fermé devant lui.
<break time="2s"/>
Les Serviteurs de Mordraid ouvrent les yeux et regardent autour d’eux afin de reconnaître tous les agents du Mal.
<break time="4s"/>
Les Serviteurs de Mordraid ferment les yeux.
<break time="2s"/>
Serviteurs de Mordraid, levez le pouce afin que Merlin puisse vous identifier.
<break time="1s"/>
Merlin, ouvre les yeux et contemple les agents du Mal.
<break time="4s"/>
Serviteurs de Mordraid, baissez le pouce et gardez le bras tendu, poing fermé.
Merlin, ferme les yeux.
<break time="2s"/>
Tout le monde ouvre les yeux.
```
